Why this matters

Every formula is a machine you can run backward.

Scenario 1: Topping up phone credit

Your final balance is g = 3 + h after a RM3 bonus. If you know the final balance, transposition finds your original balance: h = g - 3.

Scenario 2: Sharing a class-event cost

A class fund uses (R - C) / N = P, where P is each student's share. If you know P, N, and C, transposition finds the total collected amount: R = PN + C.

1. Locate the subject

The subject is the variable you want alone, usually on the left.

2. Undo in reverse order

Use inverse operations: add ↔ subtract, multiply ↔ divide, square ↔ square root.

3. Keep both sides balanced

Whatever you do to one side of the equation, do to the other side too.

Concept 1

Inverse operations: undo the last action first.

Think of the target variable as being trapped inside operations. In the workbook, g = 3 + h [h] means h is hidden behind a +3 bonus.

Start g = 3 + h
Undo +3 − 3
Balanced move g - 3 = 3 + h - 3
Simplify 3 - 3 = 0
Subject alone h = g - 3

Tips and checks

Reliable habits for error-free transposition.

Work outside to inside.

If the formula is y = 3x + 8, remove +8 before removing ×3.

Use inverse operations.

Addition is undone by subtraction. Multiplication is undone by division. Squaring is undone by square root.

Protect brackets.

Write (y - 5) / 3, not y - 5 / 3. Those are different expressions.

Common mistake: changing only one side.

From x + 6 = y, the correct move is x = y − 6, because the −6 happens to both sides.

Common mistake: treating multiplication like addition.

From 5x = y, write x = y / 5, not x = y − 5.

Edge cases to remember.

Never divide by zero. If a coefficient is 0, that variable cannot be isolated by division. If a square root appears, the real-number answer needs a non-negative radicand.
