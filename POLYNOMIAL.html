➕ Addition Formula

(ax² + bx + c) + (dx² + ex + f) = (a+d)x² + (b+e)x + (c+f)
Cara kerja: Tambah like terms sahaja
  • x² terms dengan x² terms
  • x terms dengan x terms
  • Constants dengan constants

📝 Contoh: (3x² + 5x + 2) + (2x² + 3x + 7)

Group like terms:

(3x² + 2x²) + (5x + 3x) + (2 + 7)

= 5x² + 8x + 9

➖ Subtraction Formula

(ax² + bx + c) - (dx² + ex + f) = (a-d)x² + (b-e)x + (c-f)
⚠️ PENTING: Tukar SEMUA signs dalam bracket kedua!
  • + jadi -
  • - jadi +
  • Lepas tu baru tambah macam biasa

📝 Contoh: (5x² + 7x + 3) - (2x² + 3x + 1)

Step 1: Tukar signs

5x² + 7x + 3 - 2x² - 3x - 1

Step 2: Group like terms

(5x² - 2x²) + (7x - 3x) + (3 - 1)

Step 3: Calculate

= 3x² + 4x + 2

✖️ Polynomial Multiplication (Box Method)

Box Diagram Method

(ax + b)(cx + d) = acx² + (ad + bc)x + bd
Cara kerja: Multiply row header dengan column header
  • Top-left: ax × cx = acx²
  • Top-right: ax × d = adx
  • Bottom-left: b × cx = bcx
  • Bottom-right: b × d = bd
  • Combine middle terms: adx + bcx = (ad + bc)x

➗ Polynomial Division (Synthetic Division)

Synthetic Division Method

Untuk divide: (ax² + bx + c) ÷ (x - k)
  • Step 1: Tulis coefficients dalam row pertama
  • Step 2: Tulis k (opposite sign!) di sebelah kiri
  • Step 3: Bawa turun coefficient pertama
  • Step 4: Multiply dengan k, tulis hasil
  • Step 5: Add dengan coefficient seterusnya
  • Step 6: Repeat!

💡 Tips & Common Mistakes

⚠️ Common Mistakes

Mistake 1: Lupa Tukar Signs dalam Subtraction

Salah: (5x² + 3x) - (2x² + x) = 5x² + 3x - 2x² + x = 3x² + 4x

Betul: (5x² + 3x) - (2x² + x) = 5x² + 3x - 2x² - x = 3x² + 2x

💡 Ingat: Minus sign affect SEMUA terms dalam bracket!

Mistake 2: Salah Combine Terms

Salah: 3x² + 2x = 5x³

Betul: 3x² dan 2x are DIFFERENT terms - tak boleh combine!

💡 Ingat: Hanya combine terms dengan same power!

Mistake 3: Lupa Multiply Semua Terms dalam Bracket

Salah: 3(x² + 2x + 1) = 3x² + 2x + 1

Betul: 3(x² + 2x + 1) = 3x² + 6x + 3

💡 Ingat: Multiply dengan SEMUA terms!

✨ Helpful Tips

Tip 1: Always Write Steps Clearly

Jangan terus tulis answer. Tulis step-by-step:

  1. Group like terms
  2. Calculate each group
  3. Write final answer

Tip 2: Check Your Answer

Substitute x = 1 atau x = 2 untuk check:

If (x + 2)(x + 3) = x² + 5x + 6
Try x = 1: (1+2)(1+3) = 12, and 1² + 5(1) + 6 = 12 ✓

Tip 3: Use Color Coding

Color code different terms untuk avoid confusion:

  • ■ Blue untuk x² terms
  • ■ Orange untuk x terms
  • ■ Green untuk constants

📚 Study Strategy

Master Polynomial Operations dalam 7 Steps:

  1. Understand like terms concept (x² vs x vs constants)
  2. Practice addition dengan positive numbers first
  3. Move to subtraction (remember to change signs!)
  4. Learn bracket expansion (multiply semua terms)
  5. Master box diagram method untuk multiplication
  6. Challenge yourself dengan synthetic division
  7. Practice mixed problems until confident!
