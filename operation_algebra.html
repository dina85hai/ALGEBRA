The Cool Language of Algebra

Language uses symbols and words. Algebra also uses symbols and words to solve real problems using variables or unknowns.

Term

2a, 3c, 5b, d, -2c

Expression

2a + 3a
3b - b
4g - 2g + g

Variable

x, y, m, a, k — values that can vary.

Constant

A value that does not change.

Add & Subtract

Follow the PowerPoint method: identify like terms, rearrange, then add or subtract the coefficients.

Key rule:
ax + bx = (a + b)x
Do not combine different variables: 3x + 5y ≠ 8xy
